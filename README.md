Legend Cyber Analyzer

This package includes a responsive, dark-mode frontend and a Node.js backend with advanced scans (WHOIS, RDAP, IP geolocation, TCP port checks), optional Shodan enrichment, and Stripe checkout for Pro scans.

Quick start (local):

1. Ensure you have Node.js (v16+) installed.
2. Unzip the project and open a terminal in the project root.
3. Copy .env.example to .env and fill any API keys you want to use.
4. Run:
   npm install
   npm start
5. Open http://localhost:3000

Security & Ethics:
- Only scan systems you own or have explicit permission to scan.
- Respect terms of service of APIs (Shodan, etc).
